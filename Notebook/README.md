# Fraud Deduction Project

Summary
- This repository contains an exploratory and modeling Jupyter Notebook named `Fraud_Deduction_project.ipynb` that analyzes transaction data to detect/predict potential fraudulent deductions.

Quick Start
1. Create and activate a Python 3.10+ virtual environment.
2. Install dependencies:

   pip install -r requirements.txt

3. Launch Jupyter and open the notebook:

   jupyter notebook "Fraud_Deduction_project.ipynb"

Notes
- The notebook includes data loading, cleaning, feature engineering, EDA (visualizations), model training, evaluation, and basic hyperparameter experiments.
- If your data file(s) are large or stored externally, update the notebook cells where data paths are defined.

Recommended Environment
- Python 3.10 or newer
- A virtual environment (venv, conda) is recommended to avoid dependency conflicts.
